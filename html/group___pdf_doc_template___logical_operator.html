Detailed Description

PdfDocTemplate_LogicalOperator.

The $and, $or and $not logical operators are supported.

$and
The following example uses the $and operator:
{"$and":[{"$0_width":100},{"$0_height":140}]}
You can imply the $and operator by using a comma.
The following version of the previous example uses this alternative syntax for the $and operator:
{"$0_width":100,"$0_height":140}

$not
The following example uses the $not operator:
{"$not":{"$0_width":100}}

$or
The following example uses the $or operator:
{"$or":[{"$0_width":100},{"$0_height":140}]}
