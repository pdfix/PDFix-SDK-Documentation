◆ kTrAltUpdate

const char* kTrAltUpdate = "alt_update"
static

Set alternate description for the element.

Alternate description for the element is set in a following order:
1) If kTrMc value is set to true, alternate description is taken from the original marked content and Alt tag. If there is no marked content or kTrMc is set to false, continue to the next step.
2) If kTrChildren value is set to true, alternate description is taken from the children of the element. Currently only PdeImage is supported. All texts from PdeTexts children inside the image are joined into the output alternate description. If there are no children or kTrChildren is set to false, continue to the next step.
3) If kTrPosition value is set, alternate description is taken from the neighbour text element, This PdeText element is detected as a nearest element based on kTrPosition and :kTrDistance values.
If there is no such element or kTrPosition is set to false, continue to the next step.
4) if kTrCaption is set, alternate description is taken from the nearest caption if exists.
5) If kTrAlt value is se, alternate description is taken from this node.
Params: PdeElement Parameters
Variables: PdeElement Variables
Values: kTrMc, kTrChildren, kTrPosition, kTrDistance, kTrCaption, kTrAlt
Example:

/*
"alt_update": [
{
"query": {
"params": ["pde_image"],
"$and": [
{"$0_width": {"$lte": 60}},
{"$0_height": {"$lte": 100}}
]
},
"mc": false,
"children": true,
"position": "left",
"distance": 100,
"alt":"This is an alternate description for this image."
}
]
*/

◆ kTrElementUpdate

const char* kTrElementUpdate = "element_update"
static

Element update.

Update an element after the detection of basic elements is done.
If kTrJoin is set to false, "no join" flag for the element is set. Default value is false.
If kTrLabel is set to true, element is marked as a label. Default value is false.
If kTrArtifact is set to true, element is marked as an artifact. Default value is false.
Params: PdeElement Parameters
Variables: PdeElement Variables
Values: kTrJoin, kTrLabel, kTrArtifact
Example:

/*
"element_update": [
{
"query": {
"purpose": "Move insignificant images to artifacts",
"params": ["pde_image"],
"$and": [
{"$0_width": {"$lte": 20}},
{"$0_height": {"$lte": 20}}
]
},
"artifact": true
}
]
*/

◆ kTrTableUpdate

const char* kTrTableUpdate = "table_update"
static

Table update.

Update a table after the whole process od table detection is done.
If kTrDeleteEmpty is set to true, empty rows and columns are deleted. Default value is true.
If kTrSplit is set to true, all table elements are split to fit to cells. Default value is true.
If kTrSpanEmpty is set to true, empty cells are spanned together. Default value is true.
If kTrSpanRow is set to true, rows are joined based on the maximum row span. Default value is false.
If kTrHeader is set to true, table headers are detected. It's important for table tagging kTrTagTable. Default value is true.
If kTrJoin is set to true, all texts elements in a single cell are joined into one paragraph. Default value is false.
Params: ::kTrPdeTable
Variables: PdeElement Variables
Values: kTrDeleteEmpty, kTrSplit, kTrSpanEmpty, kTrSpanRow, kTrHeader, kTrJoin
Example:

/*
"table_update": [
{
"query": {
"params": ["pde_table"],
"$and": [{"$0_col_num": 2}]
},
"split": true,
"span_empty": true,
"span_row": false
}
]
*/

◆ kTrTagImage

const char* kTrTagImage = "tag_image"
static

Tag image.

For repurposing and accessibility purposes, a Figure element should have either an Alt entry or an ActualText entry in its structure element dictionary. If there is no such entry presented, a Figure element is tagged as an artifact by default. It can be changed with kTrArtifact value set to false.
If kTrAlt value is presented, this value replaces an alternate description for the image.
If kTrActualText value is presented, this value replaces an actual text for the image.
If kTrChildren value is set to true, all image children (texts and form fields) are properly tagged. If it's set to false, all children are tagged as artifacts.
Params: ::kTrPdeImage
Variables: PdeElement Variables
Values: kTrArtifact, kTrAlt, kTrActualText, kTrChildren
Example:

/*
"tag_image": [
{
"query": {
"params": ["pde_image"],
"$and": [
{"$0_alt": ""}
]
},
"artifact": false,
"children": true,
"alt": "Figure"
}
]
*/

◆ kTrTagTable

const char* kTrTagTable = "tag_table"
static

Tag image.

If kTrArtifact value is presented, table is tagged as an artifact.
For repurposing and accessibility purposes, a Table should have headers. By default, if there were no headers detected, a Table is split to cells and each cell is tagged individually. If kTrRow value is presented, split table to cells and use rows reading order.
If kTrCol value is presented, split table to cells and use cols reading order.
If kTrSplit value is set to false, table is not split even if headers are empty. In this case a first row or col is taken as header.
