Detailed Description

PdfDocTemplate_Variable.

Variables that can be used inside a query. There are different sets of variables:
General variables: General Variables
PdeElement variables: PdeElement Variables
Example:

/*
"pagemap": [
{
"query": {
"$and": [
{"$page_num": 1}
]
},
"header_ratio": 0.1,
"footer_ratio": 0.1
},
{
"header_ratio": 0,
"footer_ratio": 0.05,
"num_cols" : 1,
"tag_form" : 1
}
],
"pagemap_regex": [
{
"hyphen": "-$"
}
]
*/
