Detailed Description

PdfDocTemplate_ArithmeticOperator.

Following artitmetic operators are supported:
::kTrFuncSum - Summary of two numbers. Expected value types int, float.
::kTrFuncDiff - Difference of two numbers. Expected value types int, float.
::kTrFuncAbs - Absolute value of a single number. Expected value types int, float.
::kTrFuncMul - Multiply two numbers. Expected value types int, float.
::kTrFuncDiv - Devide two numbers. Expected value types int, float.
::kTrFuncMin - Get minimal value from any number of values. Expected value types int, float.
::kTrFuncMax - Get maximal value from any number of values. Expected value types int, float.
