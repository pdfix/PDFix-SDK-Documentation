Detailed Description

PdfDocTemplate_PdeElementVariable.

PdeElement variable holds specific value from the current PdeElement. It can be used only with PdeElement parameter PdeElement Parameters
This variable should start with character $ that is followed by number. E.g $0_width
Number represents the index of parameter in the array presented under a "params" node.
