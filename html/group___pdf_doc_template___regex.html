Detailed Description

PdfDocTemplate_Regex.

Regex node allows to setup general regex patterns for detection algorithm.

See also
PdfDocTemplate Configuration

"pagemap": [
{
"query": {
"$and": [
{"$page_num": 1}

},

"footer_ratio": 0.1
},
{
"header_ratio": 0,

"tag_form" : 1
}
],
"pagemap_regex": [
{
"hyphen": "-$"
}
]
*/
