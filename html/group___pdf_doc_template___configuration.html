Detailed Description

PdfDocTemplate_Configuration.

The PDFix API provide rich query functionality through a configuration file.
Configuration file format: Main parent node is kTrTemplate
Pagemap node contains default parameters and query with general conditions
Page Elements contains page independent queries for initial elements
Regex allows to setup general regex patterns for detection algorithm.
