Detailed Description

PdfDocTemplate_Pagemap.

Pagemap node allows to setup general settings and thresholds for detection algorithm.

See also
PdfDocTemplate Configuration Example:
/*
"pagemap": [
{
"header_ratio": 0,
"footer_ratio": 0,
"table_detect_isolated": 0,
"label_text_detect": 0,
"label_image_detect": 0,
"line_split": 0,
"line_spacing": 1.2,
"line_spacing_nearest": 1.2,
"line_spacing_diff": 1,
"col_num" : 1
}
]
*/
