Detailed Description

PdfDocTemplate_ComparisonOperator.

A number of different comparison operators are supported.

$eq
Equality comparison operator
The following example uses the equality comparison operator:
{"$0_width":{"$eq":100}} or {"$0_width":100}

$ne
Inequality comparator operator
The following example uses the inequality comparator operator:
{"$0_width":{"$ne":100}}

$lt
The following example uses the $lt operator, which is also referred to as the "less than" comparison operator:
{"$0_width":{"$lt":100}}

$lte
The following example uses the $lte operator, which is also referred to as the Òless than or equals" operator:\n {"$0_width":{"$lte":100}}

$gt
The following example uses the $gt operator, which is also referred to as the Ògreater than" operator:\n {"$0_width":{"$gt":100}}

$gte
The following example uses the $gte operator, which is also referred to as the Ògreater than or equals" operator:\n {"$0_width":{"$gte":100}}

$regex
Regular expression predicate
The following example uses the regular expression predicate ($regex):
{"$0_font_name":{"$regex":"Bold"}}
