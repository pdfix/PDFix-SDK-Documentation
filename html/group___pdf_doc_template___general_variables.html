Detailed Description

PdfDocTemplate_GeneralVariables.

General variable can be used without parameters. It represents general state during the processing. It contains information about the current page and the document and can be used in any query.
::kTrVarPageNum Page number [int]
::kTrVarPageWidth Page cropbox width [float]
::kTrVarPageHeight Page cropbox height [float]
::kTrVarPageRotation Page rotation [int] 90 or 180 or 270 or 0
