Detailed Description

PdfDocTemplate_Page_Elements.

Page Elements are used for initial definition of the pagemap.
Initial elements are defined for each page individually and sterts wip page number "1". Node "0" is reserved for general definitions of initial elements that exists on more pages.
Each page element contains kTrPEElements node.

See also
PdfDocTemplate Configuration Example:
/*
"0": {
"bbox":[0,0,595,842],
"elements": [
{
"type": "pde_footer",
"bbox": [0,0,612,58]
}
]
}
"1": {
"bbox":[0,0,595,842],
"elements": [
{
"type": "pde_image",
"bbox": [100,100,200,200]
}
]
}
*/
